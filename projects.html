XDA Feed User Submission Portal

feed.xda-developers.com
User submission portal for XDA's Feed app. It allows users to suggest content to be curated and published in the app, as well as allows people to 'vote' for content that's already been suggested and is 'pending'. All Material Design styles done custom. Built on Vue.js.

Homes4Queers

homes4queers.herokuapp.com
Homes4Queers is a housing app built around an existing Facebook community. As a full stack developer for the project, I continue to work with the community to bring it to life. Built on Ruby on Rails 4.2.5. Features include Google Maps API and Facebook OAuth, as well as a full admin dashboard.

TwitterBox

twitterbox.herokuapp.com
This app was an excersize to ingest the Twitter API with a specific #query and style it to specific stipulations, such as an "open-door" pure css effect, and responsive styling. Pure Javascript, small Node back end for the API OAuth, CSS and HTML.
